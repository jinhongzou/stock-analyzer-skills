# A股分红配送分析

**功能**: 获取A股历年分红配送详情，包括送转比例、现金分红、股息率、每股指标、关键日期等  
**数据源**: 东方财富 (akshare) - `stock_fhps_detail_em`

---

## 使用方式

**OpenCode 调用**:
```
skill(name="a-dividend-analyzer")
```

**命令行运行**:
```bash
python .opencode/skills/a-dividend-analyzer/scripts/main.py <A股代码>
```

### 示例

```bash
python .opencode/skills/a-dividend-analyzer/scripts/main.py 600519   # 贵州茅台
python .opencode/skills/a-dividend-analyzer/scripts/main.py 000858   # 五粮液
python .opencode/skills/a-dividend-analyzer/scripts/main.py 300073   # 当升科技
```

---

## 输出字段

| 字段 | 说明 |
|------|------|
| 报告期 | 分红对应的会计期间 |
| 业绩披露日期 | 财报披露日期 |
| 送转总比例 | 送股+转股总比例（如 10.0 = 10转10） |
| 送股比例 | 送股比例 |
| 转股比例 | 资本公积转股比例 |
| 现金分红比例 | 每股现金分红金额（元） |
| 现金分红比例描述 | 分红方案文字描述 |
| 股息率 | 现金分红股息率 |
| 每股收益 | 当期每股收益 |
| 每股净资产 | 当期每股净资产 |
| 每股公积金 | 每股资本公积金 |
| 每股未分配利润 | 每股未分配利润 |
| 净利润同比增长 | 净利润同比增速 |
| 总股本 | 公司总股本 |
| 预案公告日 | 分红预案公告日期 |
| 股权登记日 | 股权登记日期 |
| 除权除息日 | 除权除息日期 |
| 方案进度 | 当前方案进度（预案/实施分配等） |
| 最新公告日期 | 最新公告日期 |

## A股分红参考标准

| 指标 | 评价 |
|------|------|
| 连续现金分红 ≥ 10年 | 分红稳定性优秀 |
| 连续现金分红 5-10年 | 分红稳定性良好 |
| 连续现金分红 3-5年 | 分红稳定性一般 |
| 股息率 > 5% | 高股息 |
| 股息率 3%-5% | 中等股息 |
| 股息率 < 3% | 低股息 |

### 送转类型说明

| 类型 | 说明 |
|------|------|
| 送股 | 用未分配利润转增股本，需缴税 |
| 转股 | 用资本公积金转增股本，不缴税 |
| 现金分红 | 直接向股东派发现金 |
