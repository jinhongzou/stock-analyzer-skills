# Report Builder - 投资分析报告生成指南

**功能**: 调用8个分析Skills获取数据，基于统一模板生成投资分析报告  
**数据源**: akshare + Tushare Pro  
**模板**: guides/unified-template.md

---

## 🔶 重要原则

**所有数据必须严格按 Skills 获取到的原始数据提供，不要修改、遗漏或添加假设数据。**

- 财务数据：使用 `financial-analysis` 返回的全部近30年数据
- ROCE：使用 `roce-calculator` 计算的实际值
- 技术指标：使用 `technical-analyzer` 返回的实际值
- 如果某年份数据缺失（如ROCE），显示为"-"，不要填写
---

## 使用方式

**本 Skill 为报告生成指南。严格依次调用以下 Skills 获取数据后，AI 组装生成报告**

### 模板文件

统一模板位于：`./guides/unified-template.md`

包含：10章报告结构 + 8个数据源Skills映射 + AI生成流程

### 步骤1：调用分析 Skills（在 OpenCode 中执行）

```
# 在对话框中依次调用这些 Skills 获取数据
1. stock-analyzer    → 基本信息+估值   skill(name="stock-analyzer")
    → {"pe": 7.7, "pb": 1.47, "股息率": 7.5%}

2. financial-analysis → 财务数据（近30年全部）
   skill(name="financial-analysis")
   → 返回29年数据：1997-2025，包含营收/净利润/ROE/ROCE/负债率

3. shareholder-deep → 股东结构
   skill(name="shareholder-deep")
   → {"集中度": 37.9%, "减持": 5次}

4. technical-analyzer → 技术指标   skill(name="technical-analyzer")
   → {"rsi": 78.24, "趋势": "震荡"}

5. market-analyzer → 市场环境
   skill(name="market-analyzer")
   → {"pe": 34.25, "趋势": "牛市"}

6. roce-calculator → ROCE趋势（实际计算值，不是财务分析中的近似值）
   skill(name="roce-calculator")
   → {"2025": 19.12%, "2024": 20.1%, ...}

7. a-dividend-analyzer → 分红历史
   skill(name="a-dividend-analyzer")
   → {"连续": 65年}

8. risk-analysis → 风险+分位数   skill(name="risk-analysis")
   → {"分位数": 94.4%}
```

**严格按 Skills 返回的数据生成报告，不要修改**
- 财务表必须包含 `financial-analysis` 返回的全部年份（近10年）
- ROCE 取 `roce-calculator` 返回的实际计算值
- 技术指标取 `technical-analyzer` 返回的实际值
- 如果某数据缺失，显示"-"，不要填写
### 步骤2：AI 组装报告

**严格按 Skills 返回的原始数据生成报告，不要修改、遗漏或填充假设值。**

收集上述 Skills 返回的数据后，AI 自动组装生成 Markdown 报告，包含：
- 10章标准结构
- 6维度综合评分
- 投资建议

### 步骤3：质量复核（必需）

**报告生成后必须按照 `guides/data-objectivity-review-guide.md` 逐项审核数据准确性和观点客观性。**

- 参照 `guides/data-objectivity-review-guide.md` 检查数据转录是否准确
- 检查每个观点是否有数据支撑
- 确认投资建议与评分逻辑自洽
- 审核通过后方可交付

- 股票代码 -> 股票名称
- 分析日期
- 数据源确认
### 步骤2：数据采集（按顺序执行）

|--------|------|-------|
| 基本信息/行情/估值 | Tushare Pro | stock-analyzer |
| 财务指标 | Tushare | financial-analysis |
| 技术指标 | 新浪财经 | technical-analyzer |
| 新闻风险 | 东方财富 | risk-analysis |
| 股东持股 | Tushare | shareholder-deep |
| ROCE趋势 | Tushare | roce-calculator |
| 分红历史 | Tushare | a-dividend-analyzer |
| 市场环境 | 乐咕 | market-analyzer |
| 历史分位数 | akshare | risk-analysis |

### 步骤3：核心分析
- 行情分析
- 业务分析
- 大股东深度分析
- 增减持分析
### 步骤4：数据验证（可选）
- PDF财报数据校验
- 多源数据交叉验证

### 步骤5：市场环境分析
- 大盘走势
- 行业轮动

### 步骤6：历史分位数分析
- 3月/1年/3年/5年分位数
- 操作建议

### 步骤7：综合评分
- 6维度打分
- 总分 + 评级

### 步骤8：报告组装
- 10章结构填充
- 图表生成

## 报告结构（10章）

| 章节 | 内容 |
|------|------|
| 一、公司概览 | 基本信息/核心资产 |
| 二、近期行情 | 今日走势/近期表现/分位数分析 |
| 三、财务分析 | 营收/利润/ROE/负债 |
| 四、业务板块 | 核心业务/战略储备 |
| 五、估值分析 | PE/PB/股息率/业绩预测 |
| 六、风险因素 | 5类风险标注 |
| 七、股东结构 | 十大股东/增减持 |
| 八、综合评分 | 6维度+总分+评级 |
| 九、投资建议 | 短中长建议/目标价 |
| 十、结论 | 优势/劣势/总结 |

## 财务分析（6维度）
### 1. 偿债能力
| 指标 | 参考标准 |
|------|---------|
| 流动比率 | >2良好，>1.5一般，<1较差 |
| 速动比率 | >1良好，>0.8一般 |
| 现金比率 | >0.5良好 |

### 2. 资本结构
| 指标 | 参考标准 |
|------|---------|
| 资产负债率 | <50%安全，50-70%警戒，>70%危险 |
| 产权比率 | 越低越安全 |

### 3. 盈利能力
| 指标 | 参考标准 |
|------|---------|
| ROE | >15%优秀，>10%良好 |
| ROA | >5%良好 |
| 毛利率 | 越高越强 |

### 4. 成长能力
| 指标 | 参考标准 |
|------|---------|
| 营收增长 | >0增长 |
| 净利润增长 | >0增长 |

### 5. 现金流
| 指标 | 参考标准 |
|------|---------|
| 经营现金流 | 正值健康 |
| 自由现金流 | 正值可分配 |

### 6. 风险预警
| 检查项 | 阈值 | 预警 |
|----------|------|------|
| 存贷双高 | 货币资金>30%+有息负债>20% | 🔶高 |
| 应收账款偏高 | 应收账款/营收>40% | 🟡中 |
| 盈利下滑 | 净利润同比下降>50% | 🟡中 |
| 大额商誉 | 商誉/总资产>20% | 🔶高 |

## 分位数分析
| 分位 | 信号 | 操作建议 |
|------|------|----------|
| <10% | 🟢极低 | 强烈买入 |
| 10-30% | 🟢低 | 买入信号 |
| 30-50% | 🟡偏低 | 分批建仓 |
| 50-70% | 🟡合理 | 观望 |
| 70-90% | 🔴偏高 | 考虑减仓 |
| >90% | 🔶极高 | 减仓/止损 |
