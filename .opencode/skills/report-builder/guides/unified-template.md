# 投资分析报告统一模板

> 本模板整合自 investment-report-template.md + report-generation-guide.md + report-model.md  
> 用于 AI 生成标准格式的投资分析报告

---

## 报告结构（10章）

| 章节 | 内容 | 必需字段 |
|------|------|----------|
| 一、公司概况 | 基本信息+核心资产 | 代码/名称/行业/市值 |
| 二、近期行情 | 今日走势+近期表现+分位数 | 股价/涨跌幅/PE分位 |
| 三、财务分析 | 营收/利润/ROE/负债 | 营收/净利润/ROE/负债率 |
| 四、业务板块 | 核心业务+战略储备 | 业务描述/产能 |
| 五、估值分析 | PE/PB/股息率/业绩预测 | PE/PB/股息率 |
| 六、风险因素 | 5类风险标注 | 风险列表 |
| 七、股东结构 | 十大股东+增减持 | 集中度/增减持 |
| 八、综合评分 | 6维度+总分+评级 | 评分/评级 |
| 九、投资建议 | 短中长建议+目标价 | 建议/目标价 |
| 十、结论 | 优势+劣势+总结 | 总结 |

---

## 数据来源 Skills（8个）

| 数据项 | Skill | 输出字段 |
|--------|-------|----------|
| 基本信息/估值 | stock-analyzer | pe/pb/股息率/市值 |
| 财务数据 | financial-analysis | roe/营收/净利润/负债率 |
| 股东结构 | shareholder-analysis | 集中度/增减持/质押 |
| 技术指标 | technical-analyzer | rsi/ma/趋势 |
| 市场环境 | market-analyzer | 市场pe/牛熊判断 |
| ROCE趋势 | roce-calculator | 近10年roce |
| 分红历史 | a-dividend-analyzer | 连续分红年数 |
| 风险+分位数 | risk-analysis | 新闻风险/价格分位 |

## AI 生成报告流程

1. **加载 report-builder 指南** → 读取本模板
2. **依次调用 8 个分析 Skills** → 获取数据
3. **按模板字段填充** → 生成报告
4. **质量复核** → 检查完整性

### 调用示例

```
skill(name="report-builder")
→ 获取模板结构

skill(name="stock-analyzer")
→ pe/pb/股息率

skill(name="financial-analysis")
→ roe/营收/净利润

skill(name="shareholder-analysis")
→ 集中度/增减持

skill(name="technical-analyzer")
→ rsi/ma/趋势

skill(name="market-analyzer")
→ 市场pe/牛熊

skill(name="roce-calculator")
→ 近10年roce

skill(name="a-dividend-analyzer")
→ 分红年数

skill(name="risk-analysis")
→ 新闻风险/分位数

AI 填充模板 → 生成报告
```
