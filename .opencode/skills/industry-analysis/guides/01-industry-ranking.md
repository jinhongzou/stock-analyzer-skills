# 01 - 行业排行分析

## 数据采集

```bash
python .opencode/skills/core/src/skills/industry-analysis/collect_industry_spot.py
```

## 核心指标

| 字段 | 含义 | 类型 |
|------|------|------|
| `total_sectors` | 行业总数 | int |
| `total_volume` | 全行业总成交额 | int（元） |
| `up_count` / `down_count` | 上涨/下跌行业数 | int |
| `top3_volume_ratio` | 前3大板块成交占比 | float（%） |
| `gainers_top10` | 涨幅榜 Top 10 | list |
| `losers_top10` | 跌幅榜 Top 10 | list |
| `volume_top10` | 成交额榜 Top 10 | list |

### 每个行业条目包含

| 字段 | 含义 |
|------|------|
| `name` | 行业名称 |
| `change_pct` | 涨跌幅（%） |
| `volume` | 成交额（元） |
| `leader_name` | 领涨股名称 |
| `leader_change` | 领涨股涨幅（%） |
| `stock_count` | 成分股数量 |

## 分析流程

1. 查看行业涨跌分布：上涨/下跌行业数量比例
2. 查看涨幅榜 Top 10：哪些行业领涨，涨幅是否异常集中
3. 查看跌幅榜 Top 10：哪些行业领跌，是否出现大面积回调
4. 查看成交额榜 Top 10：资金在哪些行业交投活跃
5. 计算行业集中度：前3大板块成交占比 > 50% 说明资金高度集中，有过度投机风险

## 分析要点

### 行业热度
- 上涨行业数是否占绝对多数（>80%）→ 市场整体情绪高涨
- 下跌行业数是否占绝对多数（>80%）→ 市场整体低迷
- 涨幅第一的行业涨幅是否 > 5% → 局部过热

### 行业集中度
- top3_volume_ratio < 30% → 资金分散，市场均衡
- top3_volume_ratio 30%-50% → 有一定集中度，关注热点持续性
- top3_volume_ratio > 50% → 资金高度集中，警惕风格切换

### 领涨股信号
- 领涨股涨幅是否远大于行业涨幅 → 龙头效应明显
- 领涨股是否为小盘股/题材股 → 投机属性强
- 领涨股所属板块是否有持续性 → 一日游还是趋势行情

