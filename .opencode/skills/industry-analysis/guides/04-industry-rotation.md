
## 综合交叉分析

### 1. 行业轮动判断

将行业分为四大类，观察资金在其中的流向变化：

| 类别 | 典型行业 | 风险偏好 |
|------|----------|----------|
| 🚀 进攻型 | 半导体/新能源/计算机/TMT | 高风险偏好（牛市） |
| ⚖️ 均衡型 | 机械/化工/汽车/消费电子 | 中等风险偏好 |
| 🛡️ 防御型 | 医药/食品饮料/公用事业 | 低风险偏好（熊市） |
| 🔄 周期型 | 银行/地产/煤炭/钢铁 | 特殊周期驱动 |

### 2. 三表交叉验证

| 维度 | 数据来源 | 信号含义 |
|------|----------|----------|
| 行业排行 | collect_industry_spot | 谁在涨/跌 |
| 资金流向 | collect_industry_fundflow | 钱去了哪 |
| 估值水平 | collect_industry_detail | 贵还是便宜 |

### 3. 信号组合解读

| 组合 | 含义 | 行动 |
|------|------|------|
| 涨幅榜 + 资金流入 + 估值合理 | 健康的趋势行情 | 顺势参与 |
| 涨幅榜 + 资金流出 + 估值偏高 | 诱多/赶顶 | 警惕回调 |
| 跌幅榜 + 资金流出 + 估值偏高 | 估值回归 | 回避 |
| 跌幅榜 + 资金流入 + 估值偏低 | 资金抄底 | 关注反转 |
| 涨幅榜 + 估值极低（修复逻辑） | 困境反转 | 可参与 |

### 4. 行业轮动阶段判断

| 阶段 | 特征 | 判断 |
|------|------|------|
| 早期 | 1-2个行业领涨，资金逐步流入 | 行情启动 |
| 中期 | 多个行业轮番表现，成交放大 | 主升浪 |
| 末期 | 补涨行业启动，龙头滞涨 | 警惕见顶 |
| 调整期 | 多数行业下跌，防御行业抗跌 | 市场回调 |

## 输出格式

```
行业轮动分析:
  [进攻型] XX行业(涨幅XX%, 主力+XX亿) → 活跃
  [防御型] XX行业(涨幅XX%, 主力+XX亿) → 关注
  
  风险偏好: 高/中/低
  轮动阶段: 早期/中期/末期/调整期

三表交叉验证:
  涨幅+资金流入+估值合理: XX行业 → ✅ 健康
  涨幅+资金流出+估值偏高: XX行业 → ⚠️ 背离
  跌幅+资金流入+估值偏低: XX行业 → 🔍 关注

综合判断:
  - 当前优势行业: XX（原因: XX）
  - 潜在风险行业: XX（原因: XX）
  - 行业轮动方向: 从XX向XX切换
```
