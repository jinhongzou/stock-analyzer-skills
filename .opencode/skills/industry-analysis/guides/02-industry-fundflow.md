# 02 - 行业资金流向分析

## 数据采集

```bash
python .opencode/skills/core/src/skills/industry-analysis/collect_industry_fundflow.py
```

## 核心指标

| 字段 | 含义 | 类型 |
|------|------|------|
| `total_sectors` | 统计行业总数 | int |
| `inflow_sectors_count` | 主力净流入行业数 | int |
| `outflow_sectors_count` | 主力净流出行业数 | int |
| `total_main_net_inflow` | 全市场主力净流入合计 | float（亿元） |
| `inflow_top10` | 主力净流入榜 Top 10 | list |
| `outflow_top10` | 主力净流出榜 Top 10 | list |

### 每个资金流条目

| 字段 | 含义 |
|------|------|
| `name` | 行业名称 |
| `main_net_inflow` | 主力净流入（亿元，负值为流出） |
| `change_pct` | 板块涨跌幅（%） |

## 分析流程

1. 查看资金流向概览：净流入vs净流出行业数量
2. 查看全市场主力净流入合计：正值为整体净流入，负值为整体净流出
3. 查看流入榜 Top 10：哪些行业在吸引资金，结合涨跌幅判断是否价格同步
4. 查看流出榜 Top 10：哪些行业在流失资金，判断是否板块轮动
5. 交叉验证：将资金流向与行业涨跌幅对比→价量配合还是背离

## 分析要点

### 资金流向与价格关系
- 价格上涨 + 主力净流入 → 健康的上涨，趋势可持续
- 价格上涨 + 主力净流出 → 诱多/背离信号，警惕回调
- 价格下跌 + 主力净流出 → 健康的调整/下跌趋势确认
- 价格下跌 + 主力净流入 → 资金抄底，关注反转机会

### 全市场资金流
- 全市场主力净流入 > 0 → 整体资金面偏暖
- 全市场主力净流入 < 0 → 整体资金面偏紧
- 净流入/流出行业数量比 > 3:1 → 极端情况，可能反转

### 板块轮动信号
- 前期领涨行业出现资金流出 → 可能板块轮动切换
- 前期滞涨行业出现资金流入 → 资金寻找洼地
- 防御性行业（医药/公用事业/消费）进入流入榜 → 市场风险偏好下降

## 输出格式

```
资金流概况:
  主力净流入/流出行业数: XX/XX
  全市场主力净流入合计: XX亿元

主力净流入榜 Top 5:
  1. XX行业: +XX亿元 涨跌幅=XX%
  2. ...

主力净流出榜 Top 5:
  1. XX行业: XX亿元 涨跌幅=XX%
  2. ...

结论:
  - 整体资金面: 偏暖/中性/偏紧
  - 板块轮动方向: 从XX流向XX
  - 背离信号: XX行业（价涨钱不跟/钱进价不涨）
```
