```bash

```

## 综合交叉分析

### 1. 行业定位

将行业分为四大类，判断目标行业所属类别及当前市场偏好：

| 类别 | 典型行业 | 风险偏好代表 |
|------|----------|-------------|
| 进攻型 | 半导体/新能源/计算机/TMT | 高风险偏好（牛市） |
| 均衡型 | 机械/化工/汽车/消费电子 | 中等风险偏好 |
| 防御型 | 医药/食品饮料/公用事业 | 低风险偏好（熊市） |
| 周期型 | 银行/地产/煤炭/钢铁 | 特殊周期驱动 |

### 2. 四维交叉验证

| 维度 | 数据来源 | 信号含义 |
|------|----------|----------|
| 实时行情 | stock_sector_spot (同花顺) | 该行业涨跌幅/成交额/排名 |
| 成分股明细 | stock_board_industry_cons_em (东方财富) | 板块内部分化/龙头效应 |
| 资金流向 | stock_sector_fund_flow_rank (东方财富) | 主力资金是否流入该行业 |
| 估值水平 | stock_industry_pe_ratio_cninfo (巨潮) | 该行业PE/PB在全市场的位置 |

### 3. 信号组合解读

| 组合 | 含义 | 行动 |
|------|------|------|
| 价格涨 + 资金流入 + 估值合理 | 健康上涨，趋势可持续 | 顺势参与 |
| 价格涨 + 资金流出 + 估值偏高 | 诱多/赶顶 | 警惕回调 |
| 价格跌 + 资金流出 + 估值偏高 | 估值回归 | 回避 |
| 价格跌 + 资金流入 + 估值偏低 | 资金逆势布局 | 关注反转 |
| 价格涨 + 龙头涨幅远大于均值 | 少数个股驱动，内部分化 | 关注扩散效应 |

### 4. 板块内部分化判断

| 上涨占比 | 平均涨幅 vs 龙头涨幅 | 判断 |
|----------|---------------------|------|
| >70% | 龙头略高于均值 | 板块普涨，趋势健康 |
| >50% | 龙头远高于均值 | 龙头驱动，关注跟风 |
| <50% | 龙头独涨 | 严重分化，警惕补跌 |
| <30% | 无强势个股 | 板块整体走弱 |

## 输出格式

```
行业深度分析: [行业名]

实时行情:
  [行业名] 涨跌幅=[变动]% 排名=[名次]/[总数]
  领涨股=[名称]([涨幅]%) 成交额=[金额]

成分股明细:
  共[总数]只成分股，上涨/下跌=[上涨]/[下跌]
  板块均价=[变动]%，龙头涨幅=[涨幅]%
  
  涨幅前3: [名称]([代码]) [涨幅]% / [名称] [涨幅]% / [名称] [涨幅]%
  跌幅前3: [名称]([代码]) [涨幅]% / [名称] [涨幅]% / [名称] [涨幅]%

资金流向:
  主力净流入=[金额]亿元（[流入/流出]）

估值:
  PE=[值]/中位数=[值] PB=[值]/中位数=[值]
  PE排名=[名次]/[总数]

交叉验证:
  [价格方向] + [资金方向] + [估值水位] → [健康/背离/关注]
  
综合判断:
  - 行业类别: [进攻/均衡/防御/周期]型
  - 板块内部分化: [普涨/龙头驱动/严重分化]
  - 信号一致性: [一致/背离/单一维度]
  - 操作建议: [顺势参与/警惕回调/关注反转]
```
