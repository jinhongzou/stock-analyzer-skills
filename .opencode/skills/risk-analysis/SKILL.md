# Risk Analysis - 风险综合分析

**功能**: 新闻风险评估 + 历史价格分位数分析  
**数据源**: 东方财富(新闻) + 新浪财经(行情)

---

## 使用方式

**OpenCode 调用**:
```
skill(name="risk-analysis")
```

**命令行运行**:
```bash
python .opencode/skills/core/src/skills/risk-analysis/main.py <股票代码> [新闻条数]
```

### 示例

```bash
python .opencode/skills/core/src/skills/risk-analysis/main.py 600519        # 贵州茅台，默认15条新闻
python .opencode/skills/core/src/skills/risk-analysis/main.py 600338 20     # 西藏珠峰，20条新闻
python .opencode/skills/core/src/skills/risk-analysis/main.py 002594 30     # 比亚迪，30条新闻
```

---

## 输出维度

### Step 1: 逐条新闻内容与风险评估
- 获取东方财富个股新闻
- 每条新闻展示：**日期、来源、标题、正文内容（前180字）**
- 关键词匹配命中结果：风险等级（高/中/低） + 风险类型（诚信风险/经营风险/正常）
- **AI 需根据每条的正文内容，独立评估其风险分类和等级**
- 风险统计汇总 + 诚信风险检测

### Step 2: 历史价格分位数
- 近3月/1年/3年/5年价格分位数
- 分位数明细表（10%-90%各分位价格）
- 当前价格在多周期中的位置

### Step 3: 综合风险判断
- 新闻风险评分（权重50%）+ 价格分位评分（权重50%）
- 综合评分 + A-E 评级
- 操作建议

---

## AI评估要求（必须在报告中体现）

当使用本技能生成分析报告时，AI **必须自己阅读每条新闻的正文内容**，做独立的风险评估，而非直接复述关键词匹配结果。

### 1. 逐条新闻内容评估

对于 `[Step 1]` 输出的 **每条新闻**，AI 必须：

1. **阅读正文内容**：仔细阅读每条新闻的标题和正文，理解事件实质
2. **独立分类**：用自己的判断给每条新闻打上风险等级：
   - **高风险**：财务造假、监管处罚、立案调查、退市风险、重大诉讼、核心高管被查等实质性利空
   - **中风险**：业绩下滑、股东减持、行业政策变化、债务违约、高管变动等可能影响股价的事件
   - **低风险**：日常经营、常规公告、行业新闻、正面报道等
3. **识别具体风险类型**：
   - 财务/会计类（如：利润调节、资产减值、审计问题）
   - 监管/法律类（如：立案调查、行政处罚、诉讼纠纷）
   - 经营/业务类（如：销量下滑、竞争加剧、客户流失）
   - 股东/治理类（如：大股东减持、股权质押、控制权变更）
4. **对比关键词匹配结果**：说明自己评估的等级与系统关键词匹配结果是否一致
5. **输出格式**：
   ```
   #1 标题: ...
     日期: ..., 来源: ...
     内容摘要: ...
     AI评估: [等级] → 理由说明
     系统匹配: [等级] → [关键词]
     一致性: ✅ 一致 / ⚠️ AI认为需升级/降级
   ```

### 2. 整体新闻风险评估

综合所有条目的分析结果：

| 条件 | 结论 |
|------|------|
| 1+ 条AI评估为高风险（含诚信风险） | 🔴 高风险，利空信号 |
| 1+ 条AI评估为高风险（经营类，非诚信） | 🔴 高风险，需警惕 |
| 3+ 条AI评估为中风险 | 🟡 中风险，中性偏空 |
| 1-2 条AI评估为中风险 | 🟡 中风险，关注后续 |
| 全部低风险 | 🟢 低风险，无重大隐患 |

### 3. 价格分位数评估

| 分位 | 信号 | 操作建议 |
|------|------|----------|
| < 10% | 🟢 极低 | 强烈买入信号 |
| 10-30% | 🟢 低 | 买入信号 |
| 30-50% | 🟡 偏低 | 可分批建仓 |
| 50-70% | 🟡 合理 | 观望或持有 |
| 70-90% | 🟠 偏高 | 考虑减仓 |
| > 90% | 🔴 极高 | 减仓/止损 |

- **多周期综合**: 取各周期分位平均值判断整体位置
- **输出**: 对应信号 + 操作建议

### 4. 综合评分

- 新闻风险 50% + 价格分位 50%
- 生成 A-E 评级
- 给出明确投资建议

## 风险评级

| 等级 | 说明 |
|------|------|
| 🟢 低风险 | 无明显负面新闻，价格处于合理偏低位置 |
| 🟡 中风险 | 存在经营风险新闻，或价格处于偏高位置 |
| 🔴 高风险 | 存在诚信风险新闻（财务造假/监管处罚等），或价格处于极端高位 |

**诚信风险关键词**（50+ 个）：财务造假、虚增利润、虚假记载、财务舞弊、信披违规、证监会调查、立案调查、行政处罚、内幕交易、操纵股价、退市风险警示等。
