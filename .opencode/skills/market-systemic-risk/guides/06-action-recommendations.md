# 综合预警与行动建议指南

**目的**: 汇总各维度分析结果，输出综合预警等级和具体行动建议

**数据采集命令**:
```bash
# 此步骤无独立采集脚本，基于前5步分析结果综合判断
# 如需执行完整数据采集，依次运行:
python .opencode/skills/core/src/skills/market-systemic-risk/collect_macro.py
python .opencode/skills/core/src/skills/market-systemic-risk/collect_structure.py
python .opencode/skills/core/src/skills/market-systemic-risk/collect_capital_flow.py
python .opencode/skills/core/src/skills/market-systemic-risk/collect_technical.py
python .opencode/skills/core/src/skills/market-systemic-risk/collect_history.py
```

## 核心指标

| 指标 | 来源 | 说明 |
|------|------|------|
| 综合评分 | 4维度加权 | Σ(维度信号分值×权重)×100，范围[-100, 100] |
| 预警等级 | 综合评分映射 | A(安全)→E(崩盘)，5级 |
| 信号方向 | 各维度分析 | positive/neutral/negative |

## 分析流程

```
1. 汇总前5步各维度信号方向和置信度
2. 按权重计算综合评分（宏经25%+结构25%+资金25%+技术25%）
3. 对照预警等级表映射综合评分 → 输出预警等级
4. 根据预警等级输出仓位建议、操作策略、对冲工具
5. 检查触发重新评估条件的信号变化
```

## 预警信号

综合评分映射为5级预警，各级含义及触发条件如下:

| 综合得分 | 预警等级 | 含义 | 触发条件 |
|----------|----------|------|----------|
| >60 | 🟢 A级 安全 | 无明显风险信号 | 全部维度 neutral 或 positive |
| 40-60 | 🟡 B级 关注 | 存在1-2个风险信号 | 1个维度 negative |
| 20-40 | 🟠 C级 警惕 | 存在3-4个风险信号 | 2个维度 negative |
| 0-20 | 🔴 D级 危险 | 存在5个以上风险信号 | 3-4个维度 negative |
| <0 | ⚫ E级 崩盘 | 确认进入崩盘通道 | 全部维度 negative + 历史验证匹配 |

## 行动建议

| 等级 | 仓位 | 策略 | 对冲 |
|------|------|------|------|
| A级 | 可满仓或适度加仓 | 顺势而为，享受牛市 | 80%股票+20%现金 |
| B级 | 建议降至80%以内 | 分散持仓，避免追高 | 60%股票+40%债券 |
| C级 | 建议降至50%以内 | 获利了结，保护利润 | 30%股票+50%债券+20%期权 |
| D级 | 建议降至20%以内或空仓 | 现金为王，等待企稳 | 10%股票+90%现金 |
| E级 | 空仓 | 不抄底，不接飞刀 | 100%现金 |

## 重新评估条件

触发以下条件时需重新评估预警等级:

1. **宏观经济**
   - GDP增速变化>1%
   - PMI突破或跌破50

2. **市场结构**
   - 产业资本单月净减持>500亿
   - PE分位变化>20%

3. **资金流动**
   - 北向资金连续3日同向流动（数据可用时）
   - 融资余额变化>5%

4. **技术信号**
   - 均线金叉/死叉
   - RSI从超买区回落至中性或直接跌破30
   - 成交量创历史新高/新低
