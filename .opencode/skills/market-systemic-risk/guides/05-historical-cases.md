# 历史案例验证指南

**目的**: 通过对比历史股灾案例的PE/指数/宏观环境，验证当前市场风险程度

**数据采集命令**:
```bash
python .opencode/skills/core/src/skills/market-systemic-risk/collect_history.py
```

## 核心指标

| 指标 | 数据来源 | 预警阈值 |
|------|----------|----------|
| 当前PE vs 2015股灾均值 | akshare `stock_market_pe_lg`（乐咕） | 当前PE≥2015均值（44.82） |
| 当前PE vs 2018熊市均值 | akshare `stock_market_pe_lg`（乐咕） | 当前PE≥2018均值（26.29） |
| 当前PE vs 2020疫情底 | akshare `stock_market_pe_lg`（乐咕） | 当前PE≥2020底部2倍 |
| 当前PE历史分位 | akshare `stock_market_pe_lg`（乐咕） | >90%分位 |
| 上证指数 vs 历史高低点 | akshare `stock_zh_index_daily` | 接近2015高点(5166) |

## 分析流程

```
1. 获取全历史PE数据（akshare stock_market_pe_lg）→ 计算历史均值/极值/分位
2. 提取2015股灾/2018熊市/2020疫情底PE区间 → 对比当前PE
3. 获取上证指数日线 → 对比历史关键点位（5178/2440/2646）
4. 综合各维度（宏观/估值/杠杆/技术）匹配度 → 输出综合相似度评分
```

## 预警信号

### 🟢 安全信号
- 当前PE低于历史崩盘均值
- PE处于历史<50%分位
- 指数距2015高点距离>30%

### 🟡 关注信号
- 当前PE接近2018熊市均值
- PE处于历史50-80%分位
- 指数距2015高点距离10-30%

### 🔴 危险信号
- 当前PE超过2015股灾均值
- PE>历史90%分位
- 指数接近2015高点（距离<10%）
- 同时出现多维度信号匹配（宏观恶化+估值泡沫+杠杆失控+技术超买）

## 对比分析框架

```
当前市场 vs 历史案例

1. 宏观经济对比
   - 当前GDP增速: X%
   - 2015年当时: ~7%
   - 对比: 好于/差于/相当

2. 估值水平对比
   - 当前PE分位: XX%
   - 2015股灾均值PE: 44.82
   - 对比: 好于/差于/相当

3. 杠杆水平对比
   - 当前融资余额: XX亿
   - 2015年当时: 极高（场外配资>2万亿）
   - 对比: 好于/差于/相当

4. 政策环境对比
   - 当前: 宽松/中性/收紧
   - 2015年: 收紧（去杠杆）
   - 对比: 好于/差于/相当
```

## 信号匹配度评估

| 信号匹配 | 权重 | 匹配度 |
|----------|------|--------|
| 宏观经济恶化 | 25% | 高/中/低 |
| 估值泡沫 | 25% | 高/中/低 |
| 杠杆失控 | 25% | 高/中/低 |
| 技术信号 | 25% | 高/中/低 |

**综合匹配度** = Σ(权重 × 匹配度得分)

- 匹配度>80%: 高度危险，参照历史崩盘模式
- 匹配度60-80%: 中度危险，需警惕
- 匹配度<60%: 相对安全，但保持关注
