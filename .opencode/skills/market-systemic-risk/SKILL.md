## 分析流程

```
Step 1: 宏观经济分析
  采集: python collect_macro.py
  分析: → guides/01-macro-analysis.md
  产出: GDP/CPI/PPI/PMI/就业/M2/PE估值 共27项

Step 2: 市场结构分析
  采集: python collect_structure.py
  分析: → guides/02-market-structure.md
  产出: PB估值/板块分布/股东增减持/新股破发率 共22项

Step 3: 资金流动分析
  采集: python collect_capital_flow.py
  分析: → guides/03-capital-flow.md
  产出: 北向资金/融资融券/主力资金(超大单/大单/中单/小单) 共22项

Step 4: 技术面信号
  采集: python collect_technical.py
  分析: → guides/04-technical-signals.md
  产出: MA均线/RSI/成交量/K线统计 共41项

Step 5: 历史案例验证
  采集: python collect_history.py
  分析: → guides/05-historical-cases.md
  产出: PE历史分阶段对比/指数关键点位对比 共33项

Step 6: 综合预警
  分析: → guides/06-action-recommendations.md
  产出: 综合评分 + 预警等级(A-E) + 行动建议
```

## 预警信号体系

### 风险等级

| 等级 | 符号 | 含义 | 行动建议 |
|------|------|------|----------|
| A级 | 🟢 安全 | 无明显风险信号 | 正常操作 |
| B级 | 🟡 关注 | 1-2个风险信号 | 谨慎操作 |
| C级 | 🟠 警惕 | 3-4个风险信号 | 减仓观望 |
| D级 | 🔴 危险 | 5个以上风险信号 | 清仓回避 |
| E级 | ⚫ 崩盘 | 确认进入崩盘通道 | 远离市场 |

### 核心预警指标

#### 一、宏观经济层面

| 信号 | 强度 | 条件 | 行动 |
|------|------|------|------|
| GDP增速持续下滑 | ⭐⭐⭐⭐☆ | 连续2季下降>0.5% | 警惕 |
| PMI跌破荣枯线 | ⭐⭐⭐⭐☆ | PMI<50 | 警惕 |
| 通胀失控 | ⭐⭐⭐⭐⭐ | CPI>5%或PPI>10% | 危险 |
| 失业率飙升 | ⭐⭐⭐⭐☆ | 失业率>5.5% | 警惕 |
| M1-M2剪刀差扩大 | ⭐⭐⭐☆☆ | M1同比<M2同比(企业活期少=投资意愿弱) | 关注 |

#### 二、市场结构层面

| 信号 | 强度 | 条件 | 行动 |
|------|------|------|------|
| 产业资本大规模减持 | ⭐⭐⭐⭐⭐ | 大股东集中减持(控股股东最危险) | 危险 |
| 估值严重泡沫化 | ⭐⭐⭐⭐☆ | PE/PB>历史90%分位(1/3/5年共振) | 警惕 |
| 板块高度集中 | ⭐⭐⭐☆☆ | 前3大板块占比>50% | 警惕 |
| 新股大面积破发 | ⭐⭐⭐☆☆ | 破发率>30%(Tushare new_share) | 警惕 |
| 新股涨幅过大 | ⭐⭐⭐☆☆ | 首日平均涨幅>200%(投机过热) | 关注 |

#### 三、资金流动层面

| 信号 | 强度 | 条件 | 行动 |
|------|------|------|------|
| 北向资金大幅流出 | ⭐⭐⭐⭐⭐ | 连续5日流出>100亿 | 危险 |
| 杠杆资金爆仓 | ⭐⭐⭐⭐⭐ | 融资余额一周下降>10% | 危险 |
| 主力资金持续净流出 | ⭐⭐⭐⭐☆ | 近20日净流出天数>15天 | 警惕 |
| 机构分歧 | ⭐⭐⭐⭐☆ | 超大单净买入+大单净卖出(分歧出货) | 警惕 |
| 散户接盘 | ⭐⭐⭐☆☆ | 主力净流出+小单净买入(接盘信号) | 关注 |

#### 四、技术面层面

| 信号 | 强度 | 条件 | 行动 |
|------|------|------|------|
| RSI(6)极度超买 | ⭐⭐⭐⭐⭐ | RSI(6)>90（短期回调概率极高） | 危险 |
| RSI(14)严重超买 | ⭐⭐⭐⭐☆ | RSI(14)>80 | 警惕 |
| 高位天量滞涨 | ⭐⭐⭐⭐⭐ | 量创历史新高但价不涨 | 危险 |
| 放量下跌趋势确认 | ⭐⭐⭐⭐⭐ | 下跌中成交量持续放大 | 危险 |
| 价格在20日区间顶部 | ⭐⭐⭐⭐☆ | position_20d_pct>95%(短期回调压力) | 警惕 |
| 均线死叉确认 | ⭐⭐⭐⭐⭐ | MA50下穿MA200 | 危险 |
| 连续大阴线 | ⭐⭐⭐⭐☆ | 3连阴或以上 | 警惕 |
| 量价背离 | ⭐⭐⭐⭐☆ | 创新高但缩量 | 警惕 |
| 跳空缺口跌破 | ⭐⭐⭐⭐☆ | 跌破重要支撑缺口 | 危险 |

## 注意事项

1. **多维度综合判断**：单一信号不构成预警，需多维度叠加
2. **时效性**：技术信号短期有效，宏观信号中长期有效
3. **数据局限性**：
   - 北向资金：东方财富接口自2024年3月起未更新，分析时需注明
   - 主力资金：最新数据约滞后6个月（当前到2025-11）
   - ETF份额：Tushare无对应接口，不可用
4. **阈值动态调整**：根据市场发展阶段调整阈值（如牛市中后期阈值更高）
5. **避免过度恐慌**：预警是风险管理工具，不是预测工具
6. **结合基本面**：技术面预警需结合基本面验证
