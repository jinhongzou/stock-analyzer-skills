# 股东-估值关联分析指南

分析股东结构与估值水平的关联关系。

---

## 一、关联逻辑

### 核心命题

```
股东结构 → 估值水平
    ↓
who invests → how valued
```

### 关联路径

| 股东类型 | 估值影响 | 传导机制 |
|----------|----------|----------|
| 机构持股 | 正相关 | 专业机构给溢价 |
| 外资持股 | 正相关 | 国际资金给溢价 |
| 国家队 | 负相关 | 稳定vs成长预期 |
| 个人持股 | 负相关 | 游资快进快出 |

## 三、分析维度

### 维度1：机构持股 vs PE

```
机构持股高 → PE较高
机构持股低 → PE较低
```

| 机构持股 | 典型PE | 含义 |
|------------|----------|------|
| >20% | 25-40 | 专业认可 |
| 10%-20% | 15-25 | 合理 |
| <10% | 10-15 | 低估或风险 |

### 维度2：外资 vs PE

```
外资持股高 → 享受溢价
外资持股低 → 正常估值
```

### 维度3：股东集中度 vs PB

```
集中 vs 分散 → 净资产折价/溢价
```

## 三、分析流程

### Step 1：获取数据

```python
# 股东数据
from core.shareholder import get_top_circulating_holders
holders = get_top_circulating_holders("000651")

# 估值数据
from core.stock import get_stock_profile
profile = get_stock_profile("000651")
```

### Step 2：计算指标

```python
# 机构持股比例
institution_ratio = sum(
    h["占流通股比例"] 
    for h in holders["holders"] 
    if "基金" in h["类型"] or "机构" in h["类型"]
)

# PE
pe = profile["估值"]["PE"]
```

### Step 3：关联判断

```python
if institution_ratio > 20 and pe > 25:
    reason = "机构给溢价"
elif institution_ratio < 10 and pe < 15:
    reason = "低估值或风险"
```

## 五、分析矩阵

| 股东结构 | 估值 | 综合判断 | 行动 |
|------------|------|----------|------|
| 机构多+外资多 | 高PE | 溢价 | 持有 |
| 机构少+外资少 | 低PE | 低估/风险 | 谨慎 |
| 机构多+低PE | 低PE | 价值洼地 | 关注 |
| 散户多+高PE | 高PE | 游资炒作 | 回避 |

---

## 六、注意事项

1. **行业差异**：不同行业PE差异大
2. **周期影响**：周期股PE失真
3. **成长预期**：高增长给高PE
4. **资金性质**：长线vs短线
