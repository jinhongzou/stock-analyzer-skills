## 三、分析维度

### 维度1：管理层持股结构

关键指标：
- 管理层直接持股比例
- 通过平台公司持股
- 股权激励授予

### 维度2：增持信号

| 信号 | 强度 | 说明 |
|------|------|------|
| 管理层直接增持 | ★★★★★ | 真金白银 |
| 股权激励授予 | ★★★☆☆ | 利益绑定 |
| 回购用于激励 | ★★☆☆☆ | 成本价 |
| 高管辞职后减持 | ★★★★★ | 负面信号 |

### 维度3：信心层级

| 层级 | 持股比例 | 信号强度 |
|------|----------|----------|
| 一级 | >30% | 强烈绑定 |
| 二级 | 10%-30% | 利益相关 |
| 三级 | <10% | 轻度绑定 |
| 四级 | 0% | 无绑定 |

## 三、信号判定规则

### 管理层信心信号

| 信号 | 条件 | 置信度 | 含义 |
|------|------|--------|------|
| 管理层持股>30% | 直接+间接 | ★★★★★ | 强绑定 |
| 管理层增持 | 本期增持 | ★★★★★ | 看好 |
| 回购注销 | 已完成 | ★★★★★ | 提升价值 |
| 大额股权激励 | >5% | ★★★★ | 利益绑定 |
| 小额股权激励 | <1% | ★★ | 象征性 |
| 离职减持 | 辞职后减持 | ★★★★★ | 看空 |

### 管理层 vs 大股东

```
管理层增持 ≠ 大股东增持
- 管理层增持：个人/团队信心
- 大股东增持：可能是战略投资
- 两者都增持：强烈看好
```

## 四、分析流程

### Step 1：识别管理层

```python
# 常见高管名称
executive_keywords = [
    "董事长", "总经理", "副总经理",
    "总裁", "副总裁", "CEO",
    " CFO", "财务总监", "董秘"
]
```

### Step 2：统计增持

```python
def analyze_executive_trend(holdings):
    increase_count = sum(1 for h in holdings if h["变动方向"] == "增持")
    decrease_count = sum(1 for h in holdings if h["变动方向"] == "减持")
    
    if increase_count > decrease_count * 2:
        signal = "增持"
    elif decrease_count > increase_count * 2:
        signal = "减持"
    else:
        signal = "中性"
```

### Step 3：估值方法

```python
if signal == "增持" and "完成比例" in data:
    # 回购支撑
    if data["完成比例"] > 100:
        confidence = "Strong"
    else:
        confidence = "Medium"
```

## 七、注意事项

1. **间接持股**：很多管理层通过平台持股
2. **代持风险**：可能存在代持情况
3. **激励成本**：注意授予价格vs市价
4. **离职影响**：离职减持是负面信号
