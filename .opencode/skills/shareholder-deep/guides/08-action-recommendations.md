# 行动建议指南

综合所有分析信号，生成投资行动建议。

---

## 一、综合评分体系

### 信号评分

| 信号类型 | 权重 | 最高分 |
|----------|------|--------|
| 股东结构 | 25% | 25 |
| 增减持信号 | 20% | 20 |
| 股权质押 | 15% | 15 |
| 股份回购 | 20% | 20 |
| 管理层增持 | 20% | 20 |

### 计算公式

```
总分 = (股东结构分 × 0.25) + (增减持分 × 0.2) + (质押分 × 0.15) + (回购分 × 0.2) + (管理层分 × 0.2)
```

## 二、推理过程标准格式（必须严格遵循）

### 每个分析步骤必须包含4个环节

```
【分析 Step X: 维度名称】

[数据获取]
  1. 调用XX接口获取XX数据...
  2. 获取到XX条记录

[计算推理]
  1. 计算指标A = ...
  2. 计算指标B = ...
  3. 与阈值对比判断：...

[判断]
  → 指标A结果：XX
  → 指标B结果：XX  
  → 综合判断：XX

[结论]
  - 信号方向：positive/negative/neutral
  - 置信度：★★★☆☆
  - 行动建议：买入/持有/观望/回避
```

### 步骤8：行动建议推理链

```
【分析 Step 8: 综合行动建议】

[数据获取]
  1. 收集Step1-7各维度信号结果
  2. 获取加权权重配置

[计算推理]
  1. 统计正向信号数量
  2. 统计负向信号数量
  3. 计算正向信号占比 = 正向 / (正向+负向+中性)
  4. 应用权重：股东结构×30% + 增减持×25% + 回购×20% + 其他×25%
  5. 生成综合评分

[判断]
  → 正向信号：N个
  → 负向信号：N个
  → 综合评分：XX分

[结论]
  - 信号方向：positive/negative/neutral
  - 置信度：★★★☆☆
  - 行动建议：XX
```

## 二、评级标准

| 评级 | 分数 | 行动建议 |
|------|------|----------|
| A | 80-100 | 积极关注，可买入 |
| B | 65-79 | 基本面良好，逢低布局 |
| C | 50-64 | 观望为主 |
| D | 35-49 | 风险较高，谨慎 |
| E | <35 | 回避 |

## 三、信号组合决策

### 买入信号组合

| 条件 | 行动 |
|------|------|
| 控股股东增持 + ROE>15% | 强烈买入 |
| 回购注销 + 机构增持 | 买入 |
| 国家队进入 + 外资流入 | 关注 |
| 业绩增长 + 估值合理 | 持有 |

### 持有信号组合

| 条件 | 行动 |
|------|------|
| 股权稳定 + 业绩稳定 | 持有 |
| 高分红 + 机构稳定 | 持有 |
| 估值合理 + 无风险 | 持有 |

### 卖出/回避信号

| 条件 | 行动 |
|------|------|
| 连续减持 + 估值高 | 卖出 |
| 高质押 + 大股东减持 | 卖出 |
| 股权分散 + 无主力 | 观望 |
| 财务造假嫌疑 | 回避 |

## 四、分析工具

### 信号检查清单

```
[ ] 股东集中度 > 50%?
[ ] 控股股东增持?
[ ] 机构持股 > 10%?
[ ] 无高比例质押?
[ ] 有回购计划?
[ ] 管理层增持?
[ ] ROE > 10%?
[ ] PE < 30?
```

### 决策树

```
                    ┌─ 控股股东增持?
                    │   └─ Yes → [强烈买入]
                    │
                    ├─ 连续减持?
                    │   └─ Yes → [卖出/回避]
                    │
                    ├─ 高质押>50%?
                    │   └─ Yes → [警惕]
                    │
                    ├─ 回购已完成?
                    │   └─ Yes → [支撑股价]
                    │
                    └─ 机构持股>10%?
                        └─ Yes → [关注]
```

## 六、风险提示

1. **单一信号不构成建议**：必须综合判断
2. **时效性注意**：数据可能有滞后
3. **行业差异**：不同行业标准不同
4. **市场风险**：系统性风险需单独考虑

---

## 七、注意事项

1. **不构成投资建议**：仅供参考
2. **风险自担**：投资有风险，入市需谨慎
3. **动态跟踪**：定期更新分析
4. **分散风险**：不要集中单一标的
