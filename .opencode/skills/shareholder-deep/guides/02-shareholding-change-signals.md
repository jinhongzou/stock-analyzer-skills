## 三、分析维度

### 维度1：单期变化

计算最新两期的持股变化：
```
变化量 = 本期持股 - 上期持股
变化率 = 变化量 / 上期持股 × 100%
```

### 维度2：连续性

识别连续增持/减持的期数：
```
连续增持 = 连续N期持股增加
连续减持 = 连续N期持股减少
```

### 维度3：抛售风险

检测连续减持的风险等级：
```
高风险 = 连续4期+减持 或 减持>5%
中风险 = 连续3期减持 或 减持>2%
低风险 = 偶尔减持
```

## 三、信号判定规则

### 增减持信号

| 信号 | 条件 | 置信度 | 含义 |
|------|------|--------|------|
| 大幅增持 | 增持>50% | ★★★★★ | 重大变化 |
| 增持 | 10%-50% | ★★★★ | 看好 |
| 小幅增持 | 1%-10% | ★★★ | 温和看好 |
| 持平 | ±1% | - | 中性 |
| 小幅减持 | -1%~-10% | ★★ | 谨慎 |
| 大幅减持 | <-10% | ★★★★ | 看空 |
| 连续减持 | 4期+ | ★★★★★ | 高风险 |

### 股东类型与增减持

| 股东类型 | 增持信号 | 减持信号 |
|----------|----------|----------|
| 控股股东 | 强烈看好 | 控制权变更 |
| 经销商 | 渠道信心 | 产品滞销 |
| 机构/基金 | 专业认可 | 短期博弈 |
| 国家队 | 政策底 | 市场顶 |
| 外资 | 国际看好 | 国际看空 |

## 四、分析流程

### Step 1：获取数据

```python
from core.shareholder import (
    get_holder_changes,
    detect_selling_risk
)

# 增减持统计
changes = get_holder_changes("000651", periods=5)

# 抛售风险检测
alerts = detect_selling_risk("000651", periods=5)
```

### Step 2：计算变化

```python
for item in changes:
    records = item["变动记录"]
    if len(records) >= 2:
        latest = records[-1]
        prev = records[-2]
        change = latest["持股数量"] - prev["持股数量"]
        change_pct = (change / prev["持股数量"]) * 100
```

### Step 3：风险判定

```python
# 高风险判定
if max_consecutive >= 4 or pct_change > 5:
    risk = "高"
elif max_consecutive >= 3 or pct_change > 2:
    risk = "中"
else:
    risk = "低"
```

## 六、注意事项

1. **数据滞后**：季度数据，有1-2月时滞
2. **口径一致**：对比同一季度数据
3. **被动减持**：可能是稀释，非主动
4. **增发影响**：增发会导致持股比例下降
