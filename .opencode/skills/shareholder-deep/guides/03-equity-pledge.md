## 三、分析维度

### 维度1：整体质押比例

```
总质押比例 = 全部未解押股数 / 总股本 × 100%
```

### 维度2：股东质押情况

按股东统计：
- 哪些股东在质押
- 质押比例
- 是否在解押

### 维度3：质押风险

| 风险等级 | 条件 | 说明 |
|----------|------|------|
| 高风险 | 总质押>50% | 股权可能变更 |
| 中风险 | 30%-50% | 关注预警 |
| 低风险 | <30% | 正常范围 |

---

## 三、信号判定规则

### 质押风险信号

| 信号 | 条件 | 置信度 | 含义 |
|------|------|--------|------|
| 高比例质押 | >50% | ★★★★★ | 高风险 |
| 控股股东质押 | >30% | ★★★★★ | 控制权风险 |
| 高管股权质押 | >50% | ★★★★ | 离职风险 |
| 无质押 | 0% | ★★★★ | 安全 |
| 解押进行中 | 状态变化 | ★★★ | 风险降低 |

### 质押原因分析

| 股东类型 | 常见原因 | 风险 |
|----------|----------|------|
| 控股股东 | 融资需求 | 中 |
| 机构投资者 | 杠杆投资 | 中 |
| 个人股东 | 资金需求 | 中-高 |
| 股权质押用于并购 | 战略需求 | 低 |

## 四、分析流程

### Step 1：获取数据

```python
from core.shareholder import get_pledge_data

pledge = get_pledge_data("000651")
# returns:
# {
#     "pledges": [...],
#     "summary": {
#         "总质押比例": "12.5%",
#         "质押笔数": 3,
#         "风险评估": "正常"
#     }
# }
```

### Step 2：风险判定

```python
total_ratio = float(pledge["summary"]["总质押比例"].strip("%"))

if total_ratio > 50:
    risk = "高"
elif total_ratio > 30:
    risk = "中"
else:
    risk = "低"
```

### Step 3：生成建议

```python
if risk == "高":
    action = "警惕，可能面临平仓风险"
elif risk == "中":
    action = "关注，质押可能变化"
else:
    action = "正常"
```
