# Comprehensive Analyzer

**功能**: A股股票系统性综合分析，一键输出完整投资报告  
**整合模块**: 市场环境 + ROCE + 财务健康 + 估值 + 技术分析 + 新闻风险 + 分红历史

---

## 使用方式

**OpenCode 调用**:
```
skill(name="comprehensive-analyzer")
```

**命令行运行**:
```bash
python .opencode/skills/comprehensive-analyzer/scripts/main.py <股票代码>
```

### 示例

```bash
python .opencode/skills/comprehensive-analyzer/scripts/main.py 600519   # 贵州茅台
python .opencode/skills/comprehensive-analyzer/scripts/main.py 600338   # 西藏珠峰
```

---

## 输出内容

| 章节 | 内容 | 数据源 |
|------|------|--------|
| 一、基本信息 | 股票名称/现价/涨跌幅/总市值/行业 | 雪球 |
| 二、市场环境 | 全市场平均PE/上证指数MA20/MA50/牛熊判断 | 乐咕 + 新浪 |
| 三、估值指标 | PE(动/静/TTM)/PB/股息率/每股收益 | 雪球 |
| 四、ROCE | 近10年资本回报率趋势 | 新浪财经 |
| 五、财务健康 | 流动比率/速动比率/资产负债率/ROE/自由现金流 | 新浪财经 |
| 六、技术分析 | MA50/MA200 金叉死叉 + RSI(14) | 新浪财经 |
| 七、新闻风险 | 诚信风险/经营风险关键词检测 | 东方财富 |
| 八、分红历史 | 近10年每股分红/股息率/送转比例 | 东方财富 |
| 九、综合评分 | 5维度评分（各20分）+ 综合评级（A-E） | 综合计算 |
| 十、投资建议 | 基于评分的买卖建议 + 风险提示 | 综合计算 |

---

## 评分体系

| 维度 | 满分 | 评估内容 |
|------|------|---------|
| 盈利能力 | 20 | ROCE 水平 |
| 财务安全 | 20 | 流动比率 + 资产负债率 |
| 估值合理性 | 20 | PE 水平 |
| 技术面 | 20 | 均线信号 + RSI |
| 新闻风险 | 20 | 负面新闻数量 |

| 总分 | 评级 | 含义 |
|------|------|------|
| 80-100 | A 优秀 | 积极关注，可考虑买入 |
| 65-79 | B 良好 | 基本面良好，逢低布局 |
| 50-64 | C 一般 | 观望为主，等待更好时机 |
| 35-49 | D 较差 | 风险较高，谨慎对待 |
| 0-34 | E 危险 | 回避，风险过大 |

**评分特点**:
- ROCE 趋势恶化自动扣分（近3年下降>50%）
- 流动比率权重加倍（短期偿债能力是关键风险）

---

## 风险提示

综合报告自动检测以下风险并提示：
- ROCE < 5%：盈利能力极弱
- 流动比率 < 1：短期偿债压力大
- 诚信风险新闻 > 0：存在财务真实性/信披合规问题
- PE > 50：估值偏高
- 市场处于熊市：注意系统性风险
