## 分析流程（整合自"龙虾的股票分析"框架）

### 第一步：信息采集（并行执行）

1. 获取股票 今日股价 行情
2. 获取股票 最新财报 业绩 营收 净利润
3. 获取股票 近期新闻 利好利空

### 第二步：五维评估

| 维度 | 核心指标 | 评分标准 |
|------|---------|---------|
| **基本面** | 营收/净利增速、ROE、毛利率、现金流 | 连续3年数据对比，看趋势 |
| **估值** | PE/PB/PS vs 行业均值 vs 历史区间 | 用估值锚点法 |
| **股息** | 股息率、分红率、分红连续性 | >=4%为高股息，>=6%为极高股息 |
| **技术面** | 52周高低点、支撑压力位、量价关系、均线 | 接近低点=机会，接近高点=谨慎 |
| **风险** | 行业风险、公司风险、宏观风险 | 逐条列出，评估影响程度 |

### 第三步：买入点计算（核心）

用 **估值锚点法** 确定买入区间：

1. 确定 52周价格分布 → 找中位数和分位点
2. 用当前PE倒推合理价格区间 → PE合理估值法
3. 用股息率倒算目标价格 → 股息率锚定法
4. 叠加技术面支撑位 → 价格验证
5. 多维交叉验证 → 找出多个条件同时满足的"甜蜜点"

**输出三个买入策略**：

| 策略 | 价格区间 | 适用场景 |
|------|---------|---------|
| 激进 | 52周中位数附近 | 不想等、看好短期 |
| 稳健 | 中位数到1/4分位之间 | 最推荐，性价比最优 |
| 理想 | 接近52周低点 | 等得起的底部猎手 |

### 第四步：透明化推理

**关键原则**: 买入点建议必须展示推导过程，不能只给结论。

至少包含：
1. 52周价格分布和中位数计算
2. PE倒推合理价格的计算表
3. 股息率倒算目标价的方法
4. 哪些维度交叉验证了该价格区间
5. 哪些部分是客观数据推导、哪些是经验判断（坦诚标注）

## 估值锚点法（核心方法）

### 维度1：52周价格分布法

- 52周高点（H）、低点（L）、中位数（M）
- Q1 = L + (H-L)*0.25, Q3 = L + (H-L)*0.75
- 激进：Q1~M；稳健：偏低1/4区间；理想：接近L

### 维度2：PE 合理估值法

| 行业类型 | 低估PE | 合理PE | 高估PE |
|---------|--------|--------|--------|
| 银行 | <5 | 5-7 | >8 |
| 公用事业 | <8 | 8-12 | >14 |
| 消费品（成熟） | <15 | 15-20 | >25 |
| 科技/医药 | <25 | 25-40 | >50 |

### 维度3：股息率锚定法

- 基准门槛：4%（跑赢银行理财）
- 优秀门槛：5%
- 极致门槛：6%

### 维度4：技术面验证

- 历史支撑位、均线支撑、成交密集区

### 特殊情况处理

| 类型 | 处理方式 |
|------|---------|
| 高成长股 | PEG估值（PE/增速<1为低估） |
| 周期股 | PB估值（PB<1为低估） |
| 亏损股 | PS或PB估值 |
| 新股 | 行业对标，提高风险权重 |

## 评分体系

| 维度 | 满分 | 评估内容 |
|------|------|---------|
| 盈利能力 | 20 | ROCE 水平 |
| 财务安全 | 20 | 流动比率 + 资产负债率 |
| 估值合理性 | 20 | PE 水平 |
| 技术面 | 20 | 均线信号 + RSI + Beta |
| 新闻风险 | 20 | 负面新闻数量 |

| 总分 | 评级 | 含义 |
|------|------|------|
| 80-100 | A 优秀 | 积极关注，可考虑买入 |
| 65-79 | B 良好 | 基本面良好，逢低布局 |
| 50-64 | C 一般 | 观望为主，等待更好时机 |
| 35-49 | D 较差 | 风险较高，谨慎对待 |
| 0-34 | E 危险 | 回避，风险过大 |

**评分特点**:
- ROCE 趋势恶化自动扣分（近3年下降>50%）
- 流动比率权重加倍（短期偿债能力是关键风险）

---

## 风险提示

综合报告自动检测以下风险并提示：
- ROCE < 5%：盈利能力极弱
- 流动比率 < 1：短期偿债压力大
- 诚信风险新闻 > 0：存在财务真实性/信披合规问题
- PE > 50：估值偏高
- 市场处于熊市：注意系统性风险
- Beta > 1.2：高波动风险
